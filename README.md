## This is a Spring Boot REST API about Student & Education System.
It is my portfolio site. I am working on this. If anyone find any error then feel free to notify me.

### Used Technology
  - Spring Framework
  - Spring Boot
  - MYSQL
  - Hibernate 5
  - Spring Security
  - Spring Boot devtools
  - Thymeleaf
  - HTML
  - CSS
  - JavaScript
  - JQuery
  - Etc.
